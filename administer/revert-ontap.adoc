= ONTAP -Revertierung wird mit AFX-Speichersystemen nicht unterstützt.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Zurücksetzen eines ONTAP Clusters ist der Prozess, bei dem alle Knoten auf die vorherige Hauptversion von ONTAP zurückgesetzt werden.

NetApp AFX-Speichersysteme unterstützen kein ONTAP Revert. Der Versuch, mit AFX eine Rückgängigmachung durchzuführen, kann zu Clusterinstabilität und Datenverlust führen. Sie sollten auf einem AFX-System keine Reverse-Operation durchführen.
